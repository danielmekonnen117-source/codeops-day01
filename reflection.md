\# Reflection



1\. What is the difference between Git and GitHub?

Git is a version control system used to track changes in files. GitHub is an online platform used to host and share Git repositories.



2\. What is the purpose of the terminal?

The terminal lets users interact with the computer by typing commands.



3\. What does git add do?

git add stages changes so they are ready to be committed.



4\. What is the purpose of git commit?

git commit saves a snapshot of the staged changes.



5\. Write the daily Git workflow in the correct order.

Edit files

git status

git add .

git commit -m "message"

git push
